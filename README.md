# Prasang 🌍

Prasang is a global poetry platform bringing together poetry from multiple cultures, languages, and traditions.

## Features

- 📜 Poetry collections (Shakespeare, Rumi, Iqbal)
- 🌐 Multi-language poetry
- 🎥 YouTube + Visual poetry content
- ⚡ Fast React frontend + Express backend
